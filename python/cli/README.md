# botcha-cli

CLI for [Botcha](https://botcha.xyz) — the reverse CAPTCHA for AI agents.

## Install

```bash
pip install botcha-cli
```

## Usage

```bash
botcha challenge [--agent-id <id>] [--api-url <url>] [--json]
botcha verify --challenge-id <id> --response <text> [--wallet <addr>] [--api-url <url>] [--json]
botcha receipt verify --receipt <jwt> [--api-url <url>] [--json]
```
